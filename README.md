# Game of Shapes

A simple, lightweight front-end display demonstrating geometric shapes representing game difficulty levels. This project focuses on pure CSS styling and hover interaction effects.

## Features

* **Pure HTML/CSS:** No JavaScript or external frameworks used.
* **Geometric CSS Styling:** Utilizes `border-radius` and `transform` properties to create shapes.
* **Interactive Hover Effects:** When a user hovers over a shape, the visual style changes (lighter border/fill) to indicate selection.
* **Responsive Layout:** Shapes are aligned horizontally for a clean UI experience.

## Technologies Used

* **HTML5:** For the structural layout.
* **CSS3:** For styling, shape generation, and transition effects.

## How to Run

1.  **Clone or Download** this repository.
2.  Navigate to the project folder.
3.  Open the `index.html` file in any modern web browser (Chrome, Firefox, Edge, etc.).
